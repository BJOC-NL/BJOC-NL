Waarom globaal?

Ook worden in Snap!, alleen globale variabelen met hun data opgeslagen wanneer het project opgeslagen wordt.
Meerdere scripts door het hele programma heen gaan deze lijst gebruiken, dus is het niet logisch om het aan maar één script vast te maken als een lokaal variabel.
